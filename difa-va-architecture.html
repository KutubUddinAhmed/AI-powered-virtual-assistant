Security Architecture

Authentication

  • GAS OIDC (enterprise SSO)
  • OAuth 2.0 with DPoP token binding
  • Flask session management
  • JWT token validation

Data Protection

  • AES encryption for PII (emails, IDs, names)
  • TLS/SSL (cert.pem + key.pem)
  • Azure DefaultAzureCredential
  • Secret rotation via environment variables

Access Control

  • Role-based access (Alice RBAC)
  • Category-based permissions
  • Admin/approver hierarchies
  • Request approval workflows
